# epub 阅读器

一个基于 `C++/Qt` 的简易 `epub` 阅读器，仅支持 `epub` 格式文件。

---
已实现：
- 支持 `windows` 和 `linux` 系统，`mac os` 未测试
- 打开 `epub` 文件，解压至本地
- 读取目录并可根据目录跳转至相应位置
- 可通过右下角滑动条调整字体大小
- 可通过拖动文件至窗口打开（`v0.3.0`）

---
